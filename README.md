SSMD (SMS Stealing Malware Dataset) is a curated academic dataset of Android malware samples focused on SMS interception and OTP-stealing techniques. The dataset is intended to support scientific research on two-factor authentication (2FA) threats and mobile malware analysis.

### Access Procedure
1. Submit the official access request form available on the project website.

   - SHA-256 hashes,
