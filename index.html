SSMD – SMS Stealing Malware Dataset

Academic dataset for research on Android SMS interception & OTP-stealing malware

The SSMD (SMS Stealing Malware Dataset) is a curated collection of Android malware samples designed to support scientific research on SMS interception, OTP theft, and related two-factor authentication (2FA) bypass techniques.

The SSMD dataset is the empirical foundation of the following peer-reviewed publication:

“Enhancing Two-Factor Authentication Security by Analyzing and Detecting SMS OTP-Interception Techniques in Android Malware”
IEEE Access, 2025
https://ieeexplore.ieee.org/document/11218835
The dataset is provided strictly for academic and research purposes only.

How to Obtain Access

To request access to the SSMD dataset, please complete the request form below. Each request is manually reviewed by the dataset maintainers.

  • 1. Complete and submit the access request form.
  • 2. We carefully review each submission.
  • 3. If your request is approved, you will receive an e-mail confirmation.
  • 4. The dataset will be sent to you as a CSV file containing:
    • malware sample identifiers,
    • SHA-256 hashes,
    • package name abbreviations.
